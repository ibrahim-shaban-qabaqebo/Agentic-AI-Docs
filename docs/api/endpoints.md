# API Endpoints

Complete reference for all API endpoints.

## Agents

### Create Agent

```bash
POST /v1/agents
```

### List Agents

```bash
GET /v1/agents
```

### Get Agent

```bash
GET /v1/agents/{id}
```

## CV Processing

### Process CV

```bash
POST /v1/cv/process
```

## Email

### Send Email

```bash
POST /v1/email/send
```
